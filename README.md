# 🛒 Shopping Cart System

A simple terminal-based shopping cart application built using Python. This project simulates a retail checkout experience where users can manage items they want to buy.

## ✨ Features
* **Add Items**: Add products to your cart along with their prices.
* **View Cart**: See a clean list of all items currently in your cart.
* **Remove Items**: Easily remove an item from your cart if you change your mind.
* **Total Calculation**: Automatically calculates the final bill amount before checkout.

## 🚀 How to Run the Project
1. Open your terminal in the project folder.
2. Run the command:
   ```bash
   python Shopping_Cart.py
   ```
